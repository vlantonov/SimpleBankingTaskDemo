```
## Prompt Scan: {filename}

### Checklist Results
{filled checklist from template}

### Violations Found: {count}
1. {check id}: {description}
2. ...

### Fixes Applied: {count}
1. {smell} → {fix}: {brief description}
2. ...

### Final State: {clean | {remaining} violations}
```
